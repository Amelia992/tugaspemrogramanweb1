Tentang Website Ini

Platform pembelajaran modern untuk memahami dasar-dasar pengembangan web dan teknologi terkini.

Website ini dikembangkan sebagai media pembelajaran interaktif yang dirancang khusus untuk memahami konsep fundamental dalam pengembangan web modern. Setiap fitur dirancang dengan pendekatan hands-on learning untuk memaksimalkan pemahaman praktis.

Tujuan Pembelajaran

HTML Fundamental

Menguasai struktur dan semantic HTML untuk membuat konten web yang terorganisir.

CSS Styling

Belajar design responsif dan modern dengan CSS dan framework terkini.

Hyperlink Navigation

Memahami navigasi antar halaman dan struktur website yang efisien.

Web Deployment

Praktik langsung dalam proses hosting dan deployment website.

Teknologi yang Digunakan

HTML5 CSS3 JavaScript Bootstrap 5 Responsive Design

Tentang Pembuat

Saya Gita amelia, seorang mahasiswa bersemangat dari Universitas Pamulang kelas 05TPLM004. Website ini merupakan hasil dedikasi saya dalam mata kuliah Pemrograman Web sebagai bukti pembelajaran dan eksplorasi dalam dunia web development.
